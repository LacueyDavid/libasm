# POUR EXECUTION 

## Creer un fichier objet .o:
nasm -f elf64 hello.s -o hello.o

## compiler le fichier .c sans linker:
gcc -c main.c -o main.o

## linker les deux emsenble:
gcc main.o func.o -o main

## creer un executable de code assembleur:
ld hello.o -o hello
